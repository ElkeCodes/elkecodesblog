React, Vue and Angular were not yet born.
jQuery was the hyped library that everyone recommended to use.
Hydration as we know it now, was in its early stages with the standardisation of AJAX.
The web was starting to let go of using frames and divs started to be looked at more to style your webpage.

If you wanted to build a website, you automatically looked at 3 options:

- ASP.NET: Microsoft’s framework for building dynamic websites, tightly integrated in the .NET ecosystem and Visual Studio
- JSP (and emerging Java frameworks like Spring MVC): Java’s enterprise-focused solution, often used for large-scale applications, with Spring MVC gaining traction as a more modern alternative to traditional JSP
- PHP: the alternative known for its ease of use, low cost, and widespread hosting support which made it ideal for small to medium projects and powering popular platforms like WordPress

So the choice of learning PHP at that time was an easy one to make as I was still a student without a job at that time (I was about 16 years old).




In the past, I've used a numerous amount of programming languages and technologies.
One of the first that I ever utilised, was PHP.
This was around 20 years back, so somewhere around 2006 that I was triggered to learn more about building websites.
These days we have all kinds of fancy frameworks like Vue, React, Angular, and so many others for the frontend and frameworks like NodeJS for backend.
The tech landscape was totally different.
NodeJS didn't exist yet, Internet Explorer 7 was just around the corner, half the internet had conversations on forums that used phpBB under the hood, jQuery didn't exist yet.

The web used technologies like ASP.Net, JSP's, ActiveX components, videos were shown in a Flash player, hydration was not yet that hot because AJAX was not fully standardised.
When you wanted to build a website, you had to look at all these proprietary technologies.
PHP was another option... a cheap and easy to start option.

When I first started programming PHP, it was still in version 4 and version 5 was heading our way.
I've written some PHP right until somwehere 2010 but haven't kept it with the advancements in the technology.
